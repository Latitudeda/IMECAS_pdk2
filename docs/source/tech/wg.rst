wg.py
==========
There are two different waveguides in IMECAS pdk, which are defined as follows:

#. ``Channel``
    #. ``Channel.C``

        * COR_WIDTH = 0.45 um
        * CLD_WIDTH = 4.45 um
        * TCH_WIDTH = 2 um

    #. ``Channel.O``

        * COR_WIDTH = 0.38 um
        * CLD_WIDTH = 4.38 um
        * TCH_WIDTH = 2 um

    .. image:: ../images/channel.png


#. ``Rib``
    #. ``Rib.C``

        * COR_WIDTH = 0.65 um
        * CLD_WIDTH = 8.65 um
        * TCH_WIDTH = 4 um

    #. ``Rib.O``

        * COR_WIDTH = 0.58 um
        * CLD_WIDTH = 8.58 um
        * TCH_WIDTH = 4 um

    .. image:: ../images/rib.png

    #. ``Rib.C_SEFE``

        * COR_WIDTH = 1 um / 0.45 um
        * CLD_WIDTH = 5 um / 4.45 um


    .. image:: ../images/rib_sefe.png
